# auth0-rems
Web service for modifying Auth0 user roles 

### Simple setup
```bash
npm start
```

### Docker

Build docker image

```bash
docker build -t auth0-rems .
```

Run docker image
```bash
docker run -p [CHOOSE_ANY_PORT]:3333 -d auth0-rems
```
